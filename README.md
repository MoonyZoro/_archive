# N-Body Gravitational Simulation

Интерактивный симулятор гравитационной системы N тел с 3D-визуализацией.

## Описание

Приложение моделирует гравитационное взаимодействие произвольного количества тел в трёхмерном пространстве. Каждое тело характеризуется положением (X, Y, Z), скоростью (Vx, Vy, Vz) и массой. Расчёт движения выполняется методом Эйлера с настраиваемым временным шагом.

## Возможности

- **3D-визуализация** основного пространства с траекториями тел
- **2D-проекция** на выбранную плоскость (XY, XZ, YZ)
- **Таблица данных** с координатами и скоростями тел в реальном времени
- **Автоматическое масштабирование** области отображения
- **Система сохранений** — создание, загрузка, удаление конфигураций (JSON)
- **Экспорт данных** симуляции в Excel (.xlsx) с отдельным листом для каждого тела
- **Управление симуляцией** — пауза, продолжение, сброс к начальным условиям
- **Ввод в научной нотации** — поддержка формата `1.5e10`, автоматическая замена запятых на точки

## Требования

- Python 3
- PyQt5
- NumPy
- Matplotlib
- openpyxl (для экспорта в Excel)

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python modelNtel.py
```

## Использование

1. **Главное меню** — создайте новое сохранение или выберите существующее
2. **Меню симуляции** — добавьте тела, задайте их параметры (координаты, скорости, массу) и настройте временной шаг (dt)
3. **Запуск** — нажмите "Запустить симуляцию" для начала моделирования
4. **Во время симуляции** — используйте паузу для просмотра данных и экспорта результатов в Excel
